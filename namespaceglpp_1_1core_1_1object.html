enum class glpp::core::object::image_format_t: GLenum

strong typed enum to represent different pixel formats

This enum is consumed by texture_t to specify the pixel format. The values mostly represent the equivalent OpenGL pixel formats. One exception is image_format_t::preferred, which has no corresponding OpenGL pixel format and can be used in the construction of texture_t when using an image_t for construction and the internal pixel format of the image_t shall be used by the texture_t.
